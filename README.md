<h3>C++ TCP library Linux</h3>

<h4>Requirement</h4>
cmake >= 3.16.0<br>
c++17 supports
<br><br>

``` 
git clone https://github.com/halloweeks/cpp-tcp-library.git
```
```
cd cpp-tcp-library
```

``` 
bash build.sh
```

<b>Run this multiple client handling</b>

```
./multiple_server
```

<b>Run this support only one client</b><br>

```
./single_server
```

<b>Run the client</b><br>

```
./client
```
