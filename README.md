<h3>C++ tcp multithread server</h3>

<h4>Requirement</h4>
<p>Make sure you have installed cmake, gcc, git clone</p>
cmake >= 3.10<br>
c++17 supports
<br><br>

``` 
git clone https://github.com/halloweeks/cpp-tcp-library.git
```
```
cd cpp-tcp-library
```

``` 
bash build.sh
```

```
./server.exe
```

```
./client.exe
```
